## homework 6
- Input:
  - 兩個整數a, b
  - 數值在0 ~ 99910 之間
  - 不考慮負數
  - a >= b
- Computation
  - 將a, b轉換為相對應的BCD code: a1, b1
  - 執行BCD code的加法運算 r1 = a1 + b1
  - 執行BCD code的減法運算 r2 = a1 – b1
- Output: BCD code
  - a1, b1, r1, r2
